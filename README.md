# AudioVideo-client-server
Audio/Video client-server for Raspberry Pi

### Install the server
`curl -s https://raw.githubusercontent.com/eelcohn/AudioVideo-client-server/refs/heads/main/avserver/install.sh | sudo bash`

### Install the client
`curl -s https://raw.githubusercontent.com/eelcohn/AudioVideo-client-server/refs/heads/main/avclient/install.sh | sudo bash`
